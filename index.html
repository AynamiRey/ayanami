Мои прогрммы

Visual Studio Code

Данная программа является для меня незаменимой в создании web-сайтов, в ней есть все необходимое для этого, а если чего-то и нет, то это можно с легкостью установить во внутреннем магазине.

PyCharm

Есть множество сред разработок для Python, но из всех, что я попробовал PyCharm понравился мне больше всего. В нем есть все необходимое для разработки, огромное количество предустановленных библиотек, удобное подстветка кода, гибкая настройка рабочего пространства. Ещё в PyCharm можно разрабатывать сайты, но для этого я предпочту Visual Studio Code.

Microsoft Office

Пакет программ Microsoft Office является незаменимым, как для работника, так и для студента.
